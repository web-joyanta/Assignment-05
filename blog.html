Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page as a structured tree, allowing developers to dynamically manipulate its content, structure, and styles.

How do you select on element from the Dom?

You can select an element from the DOM using document.querySelector('selector').

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling, reducing the need for multiple event listeners and improving performance.

How do you manipulate an element's attributes and style using DOM?

You can manipulate an element's attributes using element.setAttribute() or element.removeAttribute(), and its styles using element.style.property = "value".
